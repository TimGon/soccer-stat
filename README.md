## Структура проекта
src/
├── api/               # API-клиент и запросы
├── assets/            # изображения, стили
├── components/        # переиспользуемые компоненты
├── contexts/          # глобальные контексты
├── hooks/             # кастомные хуки
├── pages/             # страницы приложения
├── utils/             # вспомогательные функции
├── App.jsx
└── main.jsx

## Технологии, которые были использованы:

1. React (Vite)
2. Material UI (MUI) — библиотека компонентов
3. React Router DOM — навигация
4. Axios — HTTP-запросы
5. date-fns — работа с датами и локализация
6. ESLint и Prettier — контроль качества кода

# Установка и запуск локально

1. Клонируйте репозиторий:

## git clone https://github.com/ваш-логин/soccer-stat.git
## cd soccer-stat

2. Установите зависимости:
## npm install

3. Получите API-ключ:

3.1.Зарегистрируйтесь на football-data.org.
После регистрации на почту придёт токен (например, 1234567890abcdef).
4. Создайте файл .env в корне проекта и добавьте ключ:
VITE_FOOTBALL_DATA_API_KEY=ваш_токен

5. Запустите проект в режиме разработки:

## npm run dev

Приложение будет доступно по адресу http://localhost:5173.
